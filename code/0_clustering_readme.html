Scripts explanation

All the scripts are written in python 3.8. To run the script, please lunch a python tools like Anaconda or directly run "python xx.py"


  1. Frequent subgraphs mining [code]
  2. Convert results from gSpan to graph edge list and node list [code]
  3. Measure pair-wise similarity value between graphs in each scene dataset [code]
  4. Select the highest similarity one from pair-wise measurement and form dataset for clustering [code]
  5. Cluster games into groups using K-Means with PCA model. The result is viewed in a 3D plot [code]
